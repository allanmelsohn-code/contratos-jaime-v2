📌 Regras importantes

Múltiplas pessoas no mesmo papel Sempre use número: RG-locatario1.jpg, RG-locatario2.jpg.
O número define a ordem de qualificação no contrato.
Para cônjuge do locatário, use RG-locatario2.jpg (segundo locatário).
RG x CNH — qual usar? Use o documento que contiver mais dados visíveis.
A CNH traz CPF impresso, o que economiza um arquivo.
O RG da nova geração (CIN) contém CPF — prefira se disponível.
Qualidade mínima de imagem Fotos com resolução ≥ 300dpi ou scan. Evitar sombras, reflexos e texto cortado.
Para PDFs, a primeira página é usada para OCR.
Arquivos aceitos: .jpg .jpeg .png .pdf
Pasta no OneDrive Crie uma pasta por contrato: Contrato-NomeLocatario-Endereço.
Coloque todos os arquivos na raiz da pasta (sem subpastas).
Compartilhe com permissão "Qualquer pessoa com o link pode visualizar".
